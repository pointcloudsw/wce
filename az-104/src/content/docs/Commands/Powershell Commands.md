title: PowerShell Commands
---
#### Spawning processes

```powershell
Invoke-Command
Invoke-Item
Run-Job
Start-Process
"&"
```
#### Installing Azure PowerShell Module
```powershell
Install-Module az
```

#### Connect to Azure Account from PowerShell
```powershell
Connect-AzAccount -UseDeviceAuthentication -TenantId 73ec7808-5b5c-4971-b28c-663b15f02e0b
```
